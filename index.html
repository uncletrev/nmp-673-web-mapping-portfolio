Florida Red Tide Events 2007-2015

Description
This map displays the Florida Red Tide Events caused by the algae Karenia brevis throughout the state of Florida from the years 2007-2015 displayed from the samples taken by the Florida Fish and Wildlife Conservation Commission and its partners. This map is the culmination of the skills and topics learned as part of the New Maps Plus graduate certificate program.

5-Year Cancer Death Rate Trends By U.S. County

Description
Created utilizing CARTO, this web-based, choropleth map identifies 5-year cancer rate trends for each county in the United States. Each county polygon is clickable and generates a popup when clicked.

Florida Fires by County and Size From 1992-2017

Description
Created utilizing CARTO, this web-based map portrays every fire that has been documented in Florida from 1992-2017, and provides data about each fire by clicking on the circular polygons that are classified by size, in acreage, of each fire.
